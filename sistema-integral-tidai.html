Sistema Para Despacho Jurídico

Desarrollo de sistema integral diseñado específicamente para despachos jurídicos, enfocado en optimizar la gestión de casos y mejorar la eficiencia operativa. Este sistema incluye funcionalidades clave como gestión de casos, calendarios integrados para seguimiento de audiencias y plazos, y notificaciones avanzadas.


Herramientas utilizadas:


  • -HTML
  • -CSS
  • -javasScript
  • -PHP
  • -Bootstrap
  • -JQuery
  • -FontAwesome
  • -SweetAlert2
  • -FullCalendar
